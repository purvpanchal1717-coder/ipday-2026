🏎️ Robo Race

Robot Specifications

• Max size: 30×30×40 cm

• Max weight: 3 kg

• Must be remote controlled

• Voltage must not exceed 12V DC

Track Details

• Straight and curved path

• Includes obstacle zones

• Ramp sections and speed breakers

• Challenging terrain for control

During the Run

• Robot can be touched if stuck

• Each touch = +5 sec penalty

• Restart from last checkpoint

• Early start = warning restart

• Robot must not drop parts

Scoring

• Fastest time (including penalties) wins

• Tie → re-run

Common Rules

• Team size: 4 members

• Only leader communicates

• No external power

• Judge decision final
